# Система учёта заказов

Это приложение для учёта клиентов, товаров и заказов. Оно позволяет управлять клиентами, товарами и создавать заказы с несколькими товарами. 
Приложение использует библиотеку **PyQt5** для создания графического интерфейса и работает с данными через таблицы.

## Возможности

- Добавление, редактирование и удаление клиентов.
- Добавление, редактирование и удаление товаров.
- Создание и редактирование товаров.
- Формирование заказов с несколькими товарами.
- Отображение информации о заказах: номер, дата, клиент, список товаров и итоговая сумма.
- Добавление тестовых данных для ускоренного тестирования функционала.
